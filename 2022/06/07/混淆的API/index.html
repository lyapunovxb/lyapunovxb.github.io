slice、split、splice、join四个API的用法

String

String.slice()

String.slice(起始的截取索引，终止的截取索引) 从起始索引开始截取字符串，直至终止索引（不包括终止索引）。

String.split()

String.split(‘指定的字符’)将这个字符串通过指定的字符，来分割成数组，如果不写则直接将该字符串的各个字符都分割

Array

Array.slice()

Array.slice(起始截取索引，终止截取索引)，同理字符串的截取，截取该数组的中的元素并返回一个新的数组，不包括终止索引位置的元素

Array.splice()

Array.splice(起始索引，要删除的数量，要插入的元素) 是在原数组的基础上，在其任意位置增加或删除元素，会改变原数组

Array.join()

Array.join(指定的字符) 可以将数组里面的所有元素，通过指定的字符串，来连接成一个字符串并返回 注意什么都不写()则会以,逗号形式来连接

String通过[索引]获取元素

字符串可以直接通过String[索引]得到该字符

isNaN()判断是不是一个not a number

如果传入的参数不是一个数字型Number，则返回true

如果参数是一个Number，则返回false

注意：对于字符串来说，他会将字符串强转为Number型，之后再判断

将一个数字拼接形成String字符型数字

对于一个Number型的数字，我们想要让其转为字符性的只需要''+num即可

但需要注意的是，对于字符型的数字进行比较时，如果位数相同则比较结果正常

如果位数不同，则只会比较第一位数字的ASCII码值，例如这里的’3’ > ‘20’ ，因为第一位的3ASCII码要大，则会直接返回true

所以我们需要在比较之前进行类型转换 parseInt()、parseFloat(),这时比较的结果就正常了

Array.sort() 排序函数的使用

对于sort函数，我们需要在其中自定义排序规则的回调函数

arr.sort((a,b)=>{ 比较规则})

对于a和b，如果我们想让a排在b的前面，即ab， 则让其返回return -1

如果我们想要a在b的后面，即ba，则让其返回return 1

当两者相等，返回return 0

该比较函数接收两个参数(a, b)，如果第一个参数a应该位于第二个参数b的前面则返回一个负数-1，如果两个参数相等，则返回0，如果第一个参数a应位于第二个参数b的之后，则返回一个正数1，以下就是一个简单的比较函数

对于纯数字的比较我们也可以直接返回a-b则为升序排列，b-a则为降序排列
